**Иру** — это продвинутый Telegram-бот, построенный на базе современных нейросетевых архитектур (**Transformer**). Он умеет вести осмысленный диалог, понимать голосовые сообщения и анализировать эмоции пользователя.

## 🛠️ Техническое описание

### 1. Используемый API
Для взаимодействия с мессенджером используется **Telegram Bot API** через библиотеку `pyTelegramBotAPI`. 
- **Описание**: Позволяет отправлять и получать сообщения, обрабатывать голосовые данные, устанавливать реакции на сообщения (Message Reactions) и отображать статусы (typing).

### 2. Базовая модель ИИ
В качестве «мозга» бота выступает семейство моделей **Qwen2.5** (в частности, версия **72B-Instruct**), доступная через `Hugging Face Inference API`. 

### 3. Модель трансформера и решаемые задачи
Проект базируется на архитектуре **Transformer (Self-Attention)**. 
- **Используемая модель**: `Qwen2.5-72B-Instruct`.
- **Решаемые задачи**: 
    - **Генерация ответов**: Построение связного и контекстуального диалога с пользователем.
    - **Классификация текста (Sentiment Analysis)**: Определение эмоционального окраса сообщений пользователя для автоматической установки соответствующих реакций (эмодзи).
    - **Обработка контекста**: Хранение и использование истории последних 14 сообщений для поддержания нити разговора.

## 🚀 Быстрый запуск

### 1. Клонирование репозитория
```bash
git clone https://github.com/rusln4/chat_ai_tgbot.git
cd chat_ai_tgbot
```

### 2. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 3. Настройка переменных окружения
Создайте файл `.env` в корне проекта и добавьте ваши токены:
```env
BOT_TOKEN=ваш_токен_телеграм_бота
DEEPSEEK_TOKEN=ваш_токен_hugging_face
```

### 4. Запуск
```bash
python3 run.py
```
