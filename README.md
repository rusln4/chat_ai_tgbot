# 🤖 **Иру — Отказоустойчивый ИИ-Ассистент для Telegram**

</div>

**Иру** — это продвинутый Telegram-бот, спроектированный как интеллектуальный и надежный собеседник. Он построен на базе современных нейросетевых архитектур (Transformer) и оснащен многоуровневой системой отказоустойчивости, что гарантирует его работоспособность даже при сбоях внешних сервисов.

Бот способен вести осмысленный диалог, понимать голосовые сообщения, адаптироваться к стилю общения пользователя и анализировать его настроение.

### ✨ Ключевые возможности

-   **🗣️ Голосовое управление:** Иру понимает голосовые сообщения благодаря интеграции с Google Speech-to-Text API.
-   **🧠 Контекстная память:** Бот использует механизм "скользящего окна" (помнит последние 10 сообщений) для поддержания связного и осмысленного диалога.
-   **🛡️ Отказоустойчивость:** Уникальная трехуровневая архитектура ИИ гарантирует ответ пользователю даже при недоступности основной модели.
-   **🎭 Эмоциональный интеллект:** Бот анализирует настроение пользователя и устанавливает соответствующие реакции-эмодзи на сообщения.
-   **🚀 Надежный запуск:** Специальный скрипт `run.py` использует `pkill` для предотвращения запуска дубликатов бота, обеспечивая стабильность.

### 🛠️ Техническая архитектура

#### 1. Используемые API

Для реализации проекта была использована комбинация из трех ключевых API:

1.  **Telegram Bot API**
    -   **Описание:** Это основной API для взаимодействия с мессенджером Telegram. Он позволяет боту получать сообщения от пользователей, отправлять ответы, обрабатывать медиафайлы (голосовые сообщения), а также использовать элементы UX, такие как статус «печатает» и реакции на сообщения.

2.  **Hugging Face Inference API**
    -   **Описание:** Облачный сервис от платформы Hugging Face, предоставляющий доступ к тысячам предварительно обученных моделей. В нашем проекте он используется как шлюз для доступа к **основной, наиболее качественной модели ИИ**.

3.  **OpenRouter API**
    -   **Описание:** Это API-агрегатор, который предоставляет единый интерфейс для доступа к моделям от десятков различных провайдеров (включая Google, MistralAI и др.). Мы используем его для реализации **двух эшелонов резервирования**, что обеспечивает высокую надежность бота.

#### 2. Ядро ИИ: Трехуровневая система отказоустойчивости

Это ключевая особенность проекта. Система последовательно опрашивает три эшелона моделей для гарантии ответа:

-   **Эшелон 1 (Основная модель): `Qwen/Qwen2.5-72B-Instruct`**
    > Используется через **Hugging Face API**. Это сверхмощная модель от Alibaba, выбранная за высочайшее качество генерации текста на русском языке. Является приоритетным каналом для всех запросов.

-   **Эшелон 2 (Резервная модель): `google/gemma-7b-it:free`**
    > Используется через **OpenRouter API**. Включается автоматически, если основная модель недоступна (например, из-за лимитов или сбоя на стороне Hugging Face). Модель от Google, обеспечивает хороший баланс скорости и качества.

-   **Эшелон 3 (Супер-резерв): `openrouter/auto`**
    > Используется через **OpenRouter API**. Это автоматический маршрутизатор, который сам находит любую рабочую бесплатную модель из пула OpenRouter, если предыдущие два эшелона отказали. Гарантирует ответ почти в любой ситуации.

#### 3. Модель Трансформера и Решаемые Задачи

Весь проект построен на основе нейросетевой архитектуры **Transformer**, которая использует механизм внимания (Self-Attention) для обработки текста.

-   **Используемая модель:** `Qwen/Qwen2.5-72B-Instruct` (и резервные модели, также основанные на Transformer).

-   **Решаемые задачи:**

    1.  **Генерация ответов (Text Generation):** Основная задача. Модель генерирует осмысленные и релевантные ответы на основе запроса пользователя и истории диалога.

    2.  **Обработка контекста (Context Processing):** Модель анализирует историю последних 10 сообщений («скользящее окно»), чтобы поддерживать связный и логичный разговор.

    3.  **Классификация текста (Text Classification):** Модель неявно используется для определения эмоционального окраса сообщения пользователя, что позволяет боту ставить адекватные реакции-эмодзи.

### 🚀 Быстрый запуск

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/Kasilnur/TGChatbot.git
    cd TGChatbot
    ```

2.  **Установите зависимости:**
    ```bash
    pip install -r requirements.txt
    ```

3.  **Настройте переменные окружения:**
    Создайте файл `.env` в корне проекта и добавьте в него ваши ключи API:
    ```env
    BOT_TOKEN=ВАШ_ТОКЕН_TELEGRAM
    DEEPSEEK_TOKEN=ВАШ_ТОКЕН_HUGGING_FACE
    OPENROUTER_API_KEY=ВАШ_ТОКЕН_OPENROUTER
    ```

4.  **Запустите бота:**
    ```bash
    python3 run.py
    ```
